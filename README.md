# 适用于「迷上城建」的 WorldEdit CraftScripts 曲线道路建造脚本集 Mishanguc-Tools

## 前言

本项目包含一组基于 WorldEdit CraftScript 编写的 JavaScript 脚本，主要用于辅助并简化「迷上城建」中曲线道路、标线等方块的建造与替换操作，极大提升城建效率。

**安装方法**
如需使用这些脚本，请将本目录下的所有 `.js` 文件放入您的 `worldedit/craftscripts` 文件夹中。

---

## 脚本说明

### 1. `repline.js` | 标线替换与绘制

用于将相连的线段快速替换为「迷上城建」专属的标线方块，支持直接绘制**实线**或**虚线**。

* **用法**：站在目标线段上，输入 `/cs repline <类型> [空格长度] [线条长度]`
* **参数说明**：
  * **`<类型>` (必填)**：指定标线的样式。
    * `w` ：单白线
    * `wt`：白粗线
    * `wo`：偏移白线
    * `y` ：单黄线
    * `yd`：双黄线
    * `yo`：偏移黄线
  * **`[空格长度]` / `[线条长度]` (选填)**：用于自定义虚线规格。
    * 若填写这两个参数，需确保两者数值均大于 0。
    * 若不填写这两个参数，脚本将默认绘制实线。

### 2. `dashed.js` | 实线转虚线

用于将已建好的实线快速转换为虚线。

* **用法**：站在实线的**端头**，输入 `/cs dashed [空格长度] [线条长度]`
* **参数说明**：
  * **`[空格长度]` / `[线条长度]` (选填)**：定义虚线的空白间隔与实体线条长度。
    * 填写的数值必须大于 0。
    * 若不填写这两个参数，空格与线条的长度将默认取值为 3。

### 3. `repslab.js` | 道路半砖批量处理

用于快速处理道路的坡度与高度过渡。可将选区内最高一层的「迷上城建」道路整砖替换成半砖，或将最高一层的下半砖下移半格替换为双半砖。

* **用法**：选好区域后，输入 `/cs repslab` 或 `/cs repslab -r`
* **参数说明**：
  * **`-r` (选填)**：添加此参数后，脚本将**仅精准替换道路方块**，不会影响选区内的其他无关建筑或方块。
